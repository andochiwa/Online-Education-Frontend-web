# 项目说明

基于`nuxt`的在线教育网站，使用`element-ui`做组件，axios请求获取数据

## 内容

* 幻灯片播放
* 阿里云视频点播播放视频
* 购买课程
* 注册、登录、github登录
* 课程评论功能
* 课程搜索功能

# 启动方式

```bash
git clone https://github.com/andochiwa/Online-Education-Frontend-web
cd Online-Education-Frontend-web

npm install
npm run dev
```
