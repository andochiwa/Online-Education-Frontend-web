# プロジェクト説明

`Nuxt`に基づいて開発したオンライン教育サイトであり、`element-ui`をcomponentsとして使用し、axios請求でデータを取る

## 内容

* スライドショー
* AliCloud Video on Demandで動画を再生
* コースを購入
* 登録、ログイン、Githubでログイン
* コースコメント
* コース検索

# 実行方法

```bash
git clone https://github.com/andochiwa/Online-Education-Frontend-web
cd Online-Education-Frontend-web

npm install
npm run dev
```
